# vue-router-demo

Интерактивный мини-урок по Vue Router в Vue 3:
- Базовые маршруты и `router-view`
- Динамические параметры (`/users/:id`)
- Query-параметры (`page`, `sort`)
- Программная навигация (`push`, `replace`, `back`, `forward`)
- Страница 404

## Локальный запуск

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
