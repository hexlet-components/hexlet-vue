# vue-performance-optimization-demo

Интерактивный мини-урок по оптимизации производительности во Vue 3:
- Оптимизация списков через стабильный `:key`
- Статичные блоки с `v-once`
- Ленивая загрузка компонентов (`defineAsyncComponent`, `Suspense`)
- Ограничение ререндеров через `v-memo`
- Практический чеклист оптимизации

## Локальный запуск

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
