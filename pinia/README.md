# vue-pinia-demo

Интерактивный мини-урок по Pinia в Vue 3:
- Централизованное состояние для несвязанных компонентов
- `actions` и `getters` в options store
- Асинхронные действия для загрузки данных
- Setup store (`ref` + `computed` + методы)
- Практический кейс с корзиной

## Локальный запуск

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
